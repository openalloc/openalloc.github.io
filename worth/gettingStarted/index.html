Getting Started

First steps to tracking your portfolio with FlowWorth:

  1. Create (or import) your accounts
  1. Import holdings for those accounts (e.g. from brokerage positions export)
  1. Ensure each held security is priced and assigned to an asset class
  1. Create the initial valuation snapshot using the Snapshot Builder

At least one day later...

  1. Import updated holdings for your accounts
  1. (Optionally) import transactions (e.g. from brokerage transaction export)
  1. Create the second valuation snapshot using the Snapshot Builder
  1. Review how things changed!
