Snapshots Table

Each row of the ValuationSnapshot table describes a valuation captured at a particular time.

Schema

This table can be populated from the openalloc/valuation/snapshot schema.

The attributes for snapshot can be modified manually in the UI.

Typically snapshot rows are generated indirectly through import of brokerage export files. However, a comma- or tab-delimited file adhering to the following schema can also be imported through drag and drop or the Import menu:

NameTypeIsRequiredIsKeyDescript
valuationSnapshotIDstringtruetrueThe unique valuation snapshot identifier.
